
# System for Environmental Monitoring in Protected Areas
**Master's Thesis Project** | Brno University of Technology (FEKT VUT)

This repository contains a demonstration of my final thesis project: an autonomous IoT system designed for long-term monitoring of acoustic and light pollution in sensitive environments.

**Key Features**:
- **Autonomous Operation**: Powered by a solar energy harvesting system.
- **Low-Power Design**: Optimized for long-term deployment in remote areas with minimal maintenance.
- **Connectivity**: Real-time data transmission via NB-IoT.
- **Sensor Precision**: Integrated high-sensitivity sensors for measuring environmental noise and light levels.

**Tech Stack**:
- **Electronics**: Custom PCB designed in KiCad.
- **Firmware**: Developed for MCU to handle sensor data and communication protocols.
- **Mechanical Design**: Protective environmental casing designed in SolidWorks.
